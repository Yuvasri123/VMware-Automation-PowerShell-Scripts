# VMware Automation PowerShell Scripts

## Overview
This repository contains PowerShell scripts used to automate VMware infrastructure and system administration tasks.

## Technologies
- PowerShell
- VMware PowerCLI
- VMware vSphere / vSAN
- Windows Server

## Scripts Included

### VMware & vSAN
- VSAN-CapacityInfo Script
- VSAN-CapacityReport Clusters Script
- VMTags Script
- 3C Graphics Enabled VM List

### Monitoring & Reports
- Fault Storage Array Status Report
- HCI Monthly Capacity Script
- MPC Monthly Capacity Report
- Backup ID Script

### System Automation
- Install Software Through SharePath
- OS Upgrade Pre Check
- OS Upgrade Post Check
- Uninstall Script for NetBackup

### Active Directory
- AD User Account Creation Script

## How to Run
1. Open PowerShell
2. Navigate to the script folder
3. Run:
   powershell
   .\script-name.ps1

## Note
Update credentials, server names, and paths before running scripts.
